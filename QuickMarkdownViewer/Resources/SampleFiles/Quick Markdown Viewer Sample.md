# Quick Markdown Viewer Sample

Read Markdown as a clean document on macOS.

Quick Markdown Viewer is designed for a simple reading workflow: open a file and focus on the rendered page, not on editing panels or development tools.

> It should feel more like Preview and less like an IDE.

---

### Bullet Points

- Fast opening for local Markdown documents
- Clean typography in light and dark modes
- Natural scrolling, selection, and copying

## Table

| Feature | Purpose |
|---|---|
| Rendered view only | Keeps attention on the document |
| One file per window | Makes reading feel straightforward |
| Native macOS app | Starts quickly and stays lightweight |
